## Setup

### 環境変数

`.env`ファイルをコピーして`.env.development`ファイルを作成してください。
その後、それぞれ必要な値を設定してください。

#### SECRET_MICROCMS_KEY

[hello-world Manage API Keys | microCMS](https://smarthr-hello-world.microcms.io/api-keys)からAPIキーを取得できます

### 起動コマンド
```
$ yarn install
$ yarn dev
```
